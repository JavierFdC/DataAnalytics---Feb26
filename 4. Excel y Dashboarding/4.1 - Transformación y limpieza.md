## 1. Diagnóstico: ¿qué problemas tiene el dataset?

Antes de limpiar, hay que identificar qué hay que corregir:

| Problema | Ejemplo en el dataset | Impacto |
|----------|-----------------------|---------|
| Espacios extra | `" Ana García"` (espacio al inicio) | Duplica el conteo de valores únicos |
| Texto en mayúsculas inconsistente | `"MIGUEL FERNANDEZ"` vs `"Miguel Fernández"` | Agrupa incorrectamente en tablas dinámicas |
| Fechas en formato incorrecto | `"07/15/2024"` (MM/DD) mezclado con `"15/07/2024"` (DD/MM) | Errores en cálculos de fechas |
| Celdas vacías | Total_Venta en blanco | Fórmulas devuelven error o resultado incorrecto |
| Filas duplicadas | Misma venta registrada dos veces | Infla todas las métricas |
| Tipos de datos incorrectos | Números almacenados como texto | Fórmulas SUMA devuelven 0 |

### Cómo hacer un diagnóstico rápido
1. **Ctrl+Fin** → ver el rango real de datos (detecta filas vacías al final)
2. **Datos → Quitar duplicados** → ver cuántos duplicados hay antes de eliminar
3. **Formato de celdas** → verificar que fechas y números tienen el tipo correcto
4. **CONTAR.SI con criterio `"* *"`** → detectar celdas con espacios internos

## 2. Tres enfoques para limpiar datos

Los siguientes enfoques no son pasos secuenciales obligatorios — son **alternativas complementarias**. Puedes usar uno, varios o todos según la situación:

| Enfoque | Cuándo usarlo |
|---------|---------------|
| **A — Fórmulas y herramientas Excel** | Correcciones puntuales sobre un fichero concreto, cuando no se repetirá el proceso |
| **B — Power Query** | El proceso de limpieza se va a repetir con nuevos datos; necesitas que los pasos queden documentados |
| **C — Copilot** | Quieres explorar el dataset en lenguaje natural, hacer un diagnóstico rápido o no sabes por dónde empezar |

---

## Enfoque A — Fórmulas y herramientas Excel

### A1. Funciones de texto

**Espacios extra — ESPACIOS()**  
Elimina espacios al inicio, al final y los dobles espacios internos.
```
=ESPACIOS(A2)
```

**Normalizar mayúsculas**
```
=NOMPROPIO(A2)   → "pedro sánchez" → "Pedro Sánchez"
=MAYUSC(A2)      → todo en mayúsculas
=MINUSC(A2)      → todo en minúsculas
```

**Combinar: quitar espacios y normalizar a la vez**
```
=NOMPROPIO(ESPACIOS(A2))
```

**Números almacenados como texto — VALOR()**  
Si una columna numérica muestra el triángulo verde de advertencia:
```
=VALOR(A2)
```
Alternativa sin fórmula: selecciona la columna → Datos → Texto en columnas → Finalizar.

**Limpiar caracteres especiales — SUSTITUIR()**
```
=SUSTITUIR(A2, "€", "")     → elimina el símbolo euro
=SUSTITUIR(A2, ".", ",")    → cambia separador decimal
```

### A2. Flash Fill (Relleno rápido)

Flash Fill detecta el patrón de lo que estás escribiendo y completa automáticamente el resto de la columna.

**Cómo usarlo:**
1. En una columna vacía adyacente, escribe el resultado deseado para la primera fila
2. Escribe el resultado para la segunda fila
3. Excel sugiere el patrón — acepta con **Enter** o fuerza con **Ctrl+E**

**Casos de uso:**
- Separar nombre y apellidos de una misma celda
- Extraer el mes de una fecha en texto
- Reformatear códigos (ej: `"V001"` → `"VENTA-001"`)

### A3. Buscar y reemplazar

**Ctrl+B** → abre el cuadro de Buscar y reemplazar

Opciones útiles:
- **Coincidir mayúsculas/minúsculas**: diferencia "Norte" de "NORTE"
- **Celdas completas**: evita reemplazar dentro de palabras más largas
- **Usar comodines**: `*` (cualquier cadena), `?` (un carácter)

**Ejemplo:** reemplazar "MIGUEL FERNANDEZ" por "Miguel Fernández"

### A4. Quitar duplicados

1. Selecciona la tabla o rango
2. **Datos → Quitar duplicados**
3. Selecciona las columnas que definen un duplicado (normalmente ID_Venta)
4. Excel muestra cuántos duplicados eliminó

> **Buena práctica:** antes de eliminar, haz una copia de seguridad o filtra primero para revisar qué filas se consideran duplicadas.

## Enfoque B — Power Query

Power Query registra cada paso de transformación como una receta. Cuando lleguen nuevos datos, basta con actualizar para que se apliquen todos los pasos automáticamente.

### Cómo acceder
**Datos → Obtener datos → Desde tabla o rango** (si los datos ya están en Excel)

### Transformaciones esenciales

| Transformación | Dónde encontrarla en Power Query |
|----------------|----------------------------------|
| Recortar espacios | Transformar → Formato → Recortar |
| Cambiar tipo de dato | Clic en el icono del encabezado de columna |
| Quitar duplicados | Inicio → Quitar filas → Quitar duplicados |
| Reemplazar valores | Transformar → Reemplazar valores |
| Dividir columna | Transformar → Dividir columna |
| Añadir columna calculada | Agregar columna → Columna personalizada |

### Cuándo elegir Power Query sobre las fórmulas
- El dataset se actualiza periódicamente con nuevos datos
- Los pasos deben quedar documentados y ser auditables
- Necesitas revertir o modificar algún paso sin rehacer todo el trabajo

## Enfoque C — Copilot

Copilot permite describir el problema en lenguaje natural y obtener sugerencias de fórmulas o acciones. No sustituye la revisión manual, pero acelera el diagnóstico.

> **Importante:** Copilot sugiere la acción pero no edita las celdas directamente. Siempre revisa el resultado antes de aceptar.

### Detectar problemas
```
"Analiza la tabla y dime si hay celdas vacías, duplicados o inconsistencias en los datos"
```
```
"¿Hay valores en la columna Vendedor que parezcan diferentes versiones del mismo nombre?"
```

### Sugerir correcciones
```
"Añade una nueva columna llamada Vendedor_limpio con los nombres de la columna Vendedor
sin espacios extra y con el formato Nombre Apellido"
```
```
"Resalta en amarillo las filas donde Total_Venta esté vacío"
```

### Identificar errores de valor
```
"¿Hay filas en las que Total_Venta no coincida con Unidades * Precio_Unitario * (1 - Descuento_% / 100)?
Muéstrame cuáles son"
```
