## 1. ¿Qué gráfico usar para cada análisis?

| Tipo de análisis | Gráfico recomendado |
|-----------------|---------------------|
| Comparar categorías | Barras o columnas |
| Evolución en el tiempo | Líneas o áreas |
| Partes de un todo | Circular (solo si hay pocas categorías, máx. 6) |
| Distribución de valores | Histograma o caja y bigotes |
| Relación entre dos variables | Dispersión (XY) |
| Desvíos positivos/negativos | Cascada (Waterfall) |
| Comparar dos series con unidades distintas | Combinado (barras + línea) |

**Regla práctica:** si tienes más de 6 categorías, no uses un gráfico circular. Usa barras horizontales ordenadas de mayor a menor.

## 2. Crear un gráfico básico

### Desde datos directos
1. Selecciona el rango de datos (incluye los encabezados)
2. **Insertar → Gráficos recomendados** → selecciona el tipo deseado
   - O usa los accesos directos del grupo Gráficos para ir directamente al tipo
3. El gráfico aparece sobre la hoja; arrástralo y redimensiónalo

### Desde una tabla dinámica (Gráfico dinámico)
1. Haz clic dentro de la tabla dinámica
2. **Analizar tabla dinámica → Gráfico dinámico**
3. Selecciona el tipo → Aceptar
4. El gráfico dinámico se actualiza automáticamente al cambiar la tabla

## 3. Gráficos básicos

### Gráfico de columnas / barras
- **Columnas**: categorías en horizontal, valores en vertical. Mejor para comparar hasta ~10 elementos.
- **Barras**: categorías en vertical. Mejor cuando los nombres son largos o hay muchas categorías.
- Usa barras apiladas para mostrar composición y total al mismo tiempo.

**Ejercicio rápido:** crea un gráfico de columnas con Total_Venta por Categoría.

### Gráfico de líneas
- Ideal para mostrar tendencias en el tiempo.
- Necesita una dimensión temporal en el eje X (fechas, meses, trimestres).
- Evita usarlo con categorías no ordenadas.

**Ejercicio rápido:** crea un gráfico de líneas con la evolución mensual de ventas.

### Gráfico circular (tarta)
- Muestra el peso porcentual de cada parte.
- Solo si las partes suman un total significativo y hay pocas categorías.
- Activa las etiquetas de porcentaje: clic derecho sobre el gráfico → Agregar etiquetas de datos → Porcentaje.

## 4. Formato y personalización

### Elementos del gráfico
Selecciona el gráfico → aparece el icono **+** (Elementos de gráfico) a la derecha:
- **Título del gráfico**: actívalo y edita el texto directamente
- **Etiquetas de datos**: muestra los valores sobre las barras/puntos
- **Leyenda**: mueve o elimina según necesidad
- **Líneas de cuadrícula**: activa/desactiva para mayor claridad

### Colores y estilos
- **Diseño → Estilos de gráfico**: paletas predefinidas
- Para personalizar colores individuales: clic sobre una barra específica (doble clic para seleccionar solo esa serie) → **Formato → Relleno**

### Título y etiquetas de ejes
- Haz clic sobre el título y escribe directamente
- Clic sobre los valores del eje → Formato de eje → ajusta escala mínima, máxima e intervalos

## 5. Gráficos avanzados

### Gráfico combinado (dos tipos en uno)
Combina columnas para los valores y una línea para una segunda métrica (ej: ventas y descuento promedio).

**Cómo crearlo:**
1. Selecciona los datos con dos series de valores
2. **Insertar → Gráficos recomendados → Todos los gráficos → Combinado**
3. Asigna el tipo de gráfico (Columna / Línea) y si usa eje secundario para cada serie
4. Activa el **eje secundario** para la serie con escala diferente

### Gráfico de cascada (Waterfall)
Muestra contribuciones positivas y negativas a un total. Ideal para análisis de desviaciones.

**Cómo crearlo:**
1. Prepara una tabla con: categoría, valor incremental (positivo o negativo)
2. **Insertar → Cascada** (disponible en Excel 2016+)

**Caso de uso:** mostrar cómo cada categoría de producto contribuye al total de ventas.

### Gráfico de dispersión (XY)
Muestra la relación entre dos variables numéricas.

**Caso de uso:** ¿hay relación entre el descuento aplicado y las unidades vendidas?

**Cómo crearlo:**
1. Selecciona dos columnas numéricas (ej: Descuento_% y Unidades)
2. **Insertar → Dispersión**
3. Añade una **línea de tendencia**: clic derecho sobre los puntos → Agregar línea de tendencia → Lineal → mostrar ecuación

### Ejercicio 1 — Gráfico de columnas
1. Crea una tabla dinámica: Filas = Categoría, Valores = Total_Venta
2. Inserta un gráfico de columnas desde esa tabla dinámica
3. Personaliza: título, color de las barras, elimina la cuadrícula horizontal

1. Crea una tabla dinámica: Filas = Fecha (agrupada por Mes), Valores = Total_Venta

3. Añade etiquetas de datos y una línea de tendencia

### Ejercicio 3 — Gráfico combinado
1. Crea una tabla con: Mes, Total_Venta, Descuento_% promedio (desde tabla dinámica)
2. Crea un gráfico combinado: columnas para Total_Venta, línea para Descuento_%
3. Activa el eje secundario para el descuento

### Ejercicio 4 — Dispersión
1. Usa los datos directos: columnas Descuento_% y Unidades
2. Crea un gráfico de dispersión
3. Añade línea de tendencia lineal y muestra el coeficiente R²
4. Interpreta: ¿más descuento implica más unidades vendidas?
