## 1. Problemas comunes de calidad de datos

Antes de analizar datos, es imprescindible detectar y corregir estos problemas:

| Problema | Ejemplo | Impacto |
|----------|---------|---------|
| Espacios extra | `" Ana García"` | Duplica el conteo de valores únicos |
| Texto en mayúsculas inconsistente | `"MIGUEL FERNANDEZ"` vs `"Miguel Fernández"` | Agrupa incorrectamente en tablas dinámicas |
| Fechas en formato incorrecto | `"07/15/2024"` (MM/DD) mezclado con `"15/07/2024"` (DD/MM) | Errores en cálculos de fechas |
| Celdas vacías | Total_Venta en blanco | Fórmulas devuelven error o resultado incorrecto |
| Filas duplicadas | Misma venta registrada dos veces | Infla todas las métricas |
| Tipos de datos incorrectos | Números almacenados como texto | Fórmulas SUM devuelven 0 |

### Cómo hacer un diagnóstico rápido
1. **Ctrl+Fin** → ver el rango real de datos (detecta filas vacías al final)
2. **Datos → Quitar duplicados** → ver cuántos duplicados hay antes de eliminar
3. **Formato de celdas** → verificar que fechas y números tienen el tipo correcto
4. **CONTAR.SI con criterio "\* \*"** → detectar celdas con espacios

## 2. Limpieza manual: funciones de texto

### Espacios extra — RECORTAR()
Elimina espacios al inicio, al final y los dobles espacios internos.
```
=RECORTAR(A2)
```

### Normalizar mayúsculas
```
=NOMPROPIO(A2)   → "pedro sánchez" → "Miguel Fernández"
=MAYUSC(A2)      → todo en mayúsculas
=MINUSC(A2)      → todo en minúsculas
```

### Combinar funciones de limpieza
Para quitar espacios y normalizar al mismo tiempo:
```
=NOMPROPIO(RECORTAR(A2))
```

### Números almacenados como texto — VALOR()
Si una columna numérica tiene el triángulo verde de advertencia:
```
=VALOR(A2)
```
O selecciona la columna → Datos → Texto en columnas → Finalizar (conversión rápida)

### Limpiar caracteres especiales — SUSTITUIR()
```
=SUSTITUIR(A2, "€", "")     → elimina el símbolo euro
=SUSTITUIR(A2, ".", ",")    → cambia separador decimal
```

## 3. Flash Fill (Relleno rápido)

Flash Fill detecta el patrón de lo que estás haciendo y completa automáticamente el resto de la columna.

**Cómo usarlo:**
1. En una columna vacía adyacente, escribe manualmente el resultado deseado para la primera fila
2. Escribe el segundo resultado
3. Excel sugiere el patrón — acepta con **Enter** o usa **Ctrl+E** para forzarlo

**Casos de uso:**
- Separar nombre y apellidos de una misma celda
- Extraer el mes de una fecha en texto
- Formatear códigos (ej: `"V001"` → `"VENTA-001"`)

---

## 4. Buscar y reemplazar

**Ctrl+H** → abre el cuadro de Buscar y reemplazar

Opciones útiles:
- **Coincidir mayúsculas/minúsculas**: diferencia "Norte" de "NORTE"
- **Celdas completas**: evita reemplazar dentro de palabras más largas
- **Usar comodines**: `*` (cualquier cadena), `?` (un carácter)

**Ejemplo:** reemplazar todas las variantes de "PEDRO SANCHEZ" por "Miguel Fernández"

---

## 5. Quitar duplicados

1. Selecciona la tabla o rango
2. **Datos → Quitar duplicados**
3. Selecciona las columnas que definen un duplicado (normalmente ID_Venta)
4. Excel muestra cuántos duplicados eliminó

> **Buena práctica:** antes de eliminar, haz una copia de seguridad o filtra primero para revisar qué filas se consideran duplicadas.

## 6. Power Query: limpieza reproducible

Power Query registra cada paso de transformación y permite repetirlo automáticamente cuando lleguen nuevos datos.

### Cómo acceder
**Datos → Obtener datos → Desde tabla o rango** (si los datos ya están en Excel)

### Transformaciones esenciales en Power Query

| Transformación | Dónde encontrarla |
|---------------|-------------------|
| Recortar espacios | Transformar → Formato → Recortar |
| Cambiar tipo de dato | clic en el icono del encabezado de columna |
| Quitar duplicados | Inicio → Quitar filas → Quitar duplicados |
| Reemplazar valores | Transformar → Reemplazar valores |
| Dividir columna | Transformar → Dividir columna |
| Añadir columna personalizada | Agregar columna → Columna personalizada |

### Ventajas frente a la limpieza manual
- Los pasos quedan registrados y son auditables
- Se puede actualizar con nuevos datos sin repetir el trabajo
- Es reversible: se puede eliminar cualquier paso

## 7. Copilot para limpieza de datos

### Detectar problemas
```
"Analiza la tabla y dime si hay celdas vacías, duplicados o inconsistencias en los datos"
```
```
"¿Hay valores en la columna Vendedor que parezcan diferentes versiones del mismo nombre?"
```

### Sugerir correcciones
```
"Añade una nueva columna llamada Vendedor_limpio con los nombres de la columna Vendedor
sin espacios extra y con el formato Nombre Apellido"
```
```
"Resalta en amarillo las filas donde Total_Venta esté vacío"
```

### Identificar outliers o errores de valor
```
"¿Hay filas en las que Total_Venta no coincida con Unidades * Precio_Unitario * (1 - Descuento_% / 100)?
Muéstrame cuáles son"
```

> **Nota:** Copilot sugiere la fórmula o la acción, pero no edita directamente las celdas. Siempre revisa antes de aceptar.
