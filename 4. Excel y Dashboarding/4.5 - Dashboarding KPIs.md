## 1. ¿Qué es un dashboard?

Un dashboard es una página de resumen visual que permite tomar decisiones a partir de los datos de un vistazo. Un buen dashboard:

- **Responde preguntas concretas** — tiene un propósito definido
- **Se entiende en menos de 30 segundos** — sin necesidad de explicación
- **Muestra solo lo esencial** — no todo lo que existe en los datos
- **Es interactivo** — permite filtrar y explorar sin editar los datos

### Los tres niveles de un dashboard

| Nivel | Qué muestra | Ejemplo |
|-------|------------|---------|
| **Estratégico** | KPIs globales del negocio | Total ventas, margen, crecimiento YoY |
| **Táctico** | Comparativas y tendencias | Ventas por región, por canal, por mes |
| **Operacional** | Detalle de transacciones | Tabla de ventas filtrables |

## 2. KPIs: definición y diseño

Un KPI (Key Performance Indicator) es una métrica que mide el progreso hacia un objetivo clave.

### Qué hace un buen KPI
- Tiene un **objetivo de referencia** (target) con el que compararse
- Indica claramente si el resultado es **positivo o negativo**
- Se puede calcular de forma consistente y repetible

### KPIs típicos en un dashboard de ventas

| KPI | Fórmula |
|-----|---------|
| Total de ventas | SUMA(Total_Venta) |
| Número de operaciones | CONTARA(ID_Venta) |
| Ticket medio | Total_Venta / Número de operaciones |
| Ventas Online vs Tienda | SUMAR.SI(Canal,"Online",Total_Venta) |
| Mejor vendedor | BUSCARV del máximo de ventas por vendedor |
| Crecimiento mensual | (Mes actual - Mes anterior) / Mes anterior |

## 3. Tarjetas de KPI en Excel

Excel no tiene un componente de "tarjeta KPI" nativo como Power BI, pero se puede construir con:

### Método 1 — Celdas con formato visual
1. Reserva un bloque de celdas (ej: 3 filas × 2 columnas)
2. Escribe el valor del KPI con fórmula (grande, negrita)
3. Escribe la etiqueta (descripción del KPI, tamaño más pequeño)
4. Aplica fondo de color y sin bordes de celda (para que parezca una tarjeta)
5. Elimina las líneas de cuadrícula en la hoja: **Vista → Líneas de cuadrícula** (desmarcar)

### Método 2 — Formato condicional para indicar estado

Añade un indicador visual de estado:
- Verde si el KPI supera el objetivo
- Amarillo si está cerca
- Rojo si está por debajo

**Cómo hacerlo:**
1. Selecciona la celda del KPI
2. **Inicio → Formato condicional → Conjuntos de iconos**
3. Elige flechas o semáforos
4. Configura los umbrales según el objetivo

### Método 3 — Sparklines (minigráficos)

Los sparklines son gráficos en miniatura que caben dentro de una celda. Ideales para mostrar la tendencia de un KPI.

1. Selecciona la celda donde quieres el sparkline
2. **Insertar → Minigráficos → Línea**
3. Selecciona el rango de datos (ej: ventas de los 12 meses)
4. Activa el marcador de punto alto: **Minigráfico → Marcar puntos → Punto más alto**

## 4. Principios de diseño visual

### Layout (distribución)

Sigue el patrón de lectura natural: de izquierda a derecha, de arriba a abajo.

```
┌─────────────────────────────────────────────────────┐
│  TÍTULO DEL DASHBOARD    │  Segmentadores / Filtros  │
├──────────┬──────────┬────┴──────────────────────────┤
│  KPI 1   │  KPI 2   │  KPI 3   │  KPI 4             │
├──────────┴──────────┴──────────┴────────────────────┤
│  Gráfico principal (grande)                          │
├─────────────────────────┬───────────────────────────┤
│  Gráfico secundario     │  Tabla / Gráfico terciario │
└─────────────────────────┴───────────────────────────┘
```

**Reglas de layout:**
- Los KPIs van en la parte superior (se ven primero)
- El gráfico más importante ocupa más espacio
- Los detalles van abajo o en pestaña aparte

### Color

- Usa una **paleta de 2-3 colores** como máximo
- Reserva el color de acento (más llamativo) para lo más importante
- Usa escala de grises para elementos secundarios y de contexto
- Evita fondos muy oscuros con texto oscuro (bajo contraste)

**Paletas recomendadas para dashboards de negocio:**
- Azul corporativo + gris + blanco
- Verde oscuro + beige + blanco
- Gris oscuro + naranja de acento + blanco

### Tipografía

- **Título principal**: Calibri o Segoe UI, 18-22pt, negrita
- **Títulos de sección**: 12-14pt, negrita o semibold
- **Etiquetas y valores**: 10-11pt, sin negrita (excepto los números grandes de KPI)
- Evita mezclar más de 2 fuentes
- Alinea siempre el texto de forma consistente

### Espacio en blanco

El espacio vacío no es espacio perdido — es claridad visual. Deja márgenes entre elementos y no rellenes cada centímetro de la hoja.

## 5. Preparar la hoja para presentación

### Desactivar elementos visuales de Excel
- **Vista → Líneas de cuadrícula**: desmarcar
- **Vista → Encabezados**: desmarcar (oculta los números de fila y letras de columna)
- **Vista → Barra de fórmulas**: opcional desmarcar en modo presentación

### Proteger el dashboard
Para que los usuarios no modifiquen la estructura accidentalmente:
1. Desbloquea las celdas donde sí quieres que interactúen (segmentadores, filtros)
2. **Revisar → Proteger hoja** → define contraseña si es necesario
3. Activa solo las opciones de interacción permitidas (ej: seleccionar celdas desbloqueadas)

### Organizar las hojas del libro
Separa los datos de la presentación:
- Hoja `Datos`: datos originales
- Hoja `Cálculos`: tablas dinámicas y fórmulas intermedias
- Hoja `Dashboard`: el panel final (solo visualización)

Oculta las hojas intermedias: clic derecho sobre la pestaña → **Ocultar**
